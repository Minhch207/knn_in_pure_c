# Thuật toán

Knn là một thuật toán học máy trong đó ta chọn k điểm gần nhất, nhãn nào xuất hiện nhiều nhất trong k điểm thì ta chọn nhãn đó là giá trị dự đoán.

# Sử dụng

source code sử dụng tếp data.inp để nhập dữ liệu và có dạng

```
[số dữ liệu n] 
[sô đặc trưng d]
[ma trận có dạng n x d]
[n nhãn]
[điểm cần dự đoán với d đặc trưng] 
```

ví dụ:
```
7
3
3 2 4
2 1 2
1 2 1
2 3 5
1 2 3
3 3 4
5 3 5
rose tulip tulip rose tulip rose rose
4 3 4
```
